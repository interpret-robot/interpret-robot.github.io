InterPreT: Interactive Predicate Learning from Language Feedback for Generalizable Task Planning

Muzhi Han1, Yifeng Zhu2, Song-Chun Zhu1, Ying Nian Wu1, Yuke Zhu2,
1University of California, Los Angeles, 2The University of Texas at Austin
RSS 2024
Paper Supplementary Code arXiv

Abstract

We present InterPreT, an LLM-powered framework that allows robots to autonomously learn symbolic predicates for long-horizon task planning. from language feedback of non-expert human users during embodied interaction. These learned predicates provide relational state abstractions over continuous states, enabling the system to also learn symbolic operators (i.e., action preconditions and effects) that can handle scenarios with varying number of objects. InterPreT compiles the learned predicates and operators into a PDDL domain file, which allows effective planning toward arbitrary in-domain goals with classical planners. In simulated and real-world robot manipulation domains, InterPreT learns from a series of simple training tasks guided by language feedback and is evaluated on a comprehensive test set featuring different levels of generalization requirements. We demonstrate that InterPreT reliably uncovers the key predicates and operators governing the environment dynamics. Notably, it achieves strong generalization performance on novel tasks significantly more complex than those in the training set. In the most challenging generalization setting, InterPreT attains success rates of 73% in simulation and 40% in the real world, substantially outperforming baseline methods.

Method

InterPreT leverages the power of GPT-4 to learn predicates by generating Python functions, which are then refined via iterative prompting. Then a cluster-then-search algorithm learns symbolic operators from collected interaction data. At test time, we utilize another GPT-4 to translate language goal specifications into symbolic goals to enable PDDL planning with the predicates and operators.
